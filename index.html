SnapElements

Webbased frontend for PowerDNS

There currently are a couple of PowerDNS frontends available, but none of them seems to be fulfilling the true needs of a DNS-administrator. This project has the intend to be a PowerDNS-administrator's best friend :).

Install

mysql -u powerdns -p powerdns < install/db.sql
